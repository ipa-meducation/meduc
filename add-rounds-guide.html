How to Add New Implementation Rounds

Step-by-step guide for expanding your mEducation Hub

Back to Converter

Quick Overview

To add new implementation rounds to your mEducation Hub, you only need to update two files:

  • rounds-config.json - Add the new round configuration
  • schools.json - Update school data (use the Map File Converter)

The Implementation Map will automatically detect and display new rounds!

Step-by-Step Process

Step 1: Update rounds-config.json

Edit the rounds-config.json file and add your new round to the implementationRounds array:

{ "implementationRounds": [ { "id": "pilot-ipa", "name": "Pilot - IPA Arm", "color": "#84d0d4", "description": "Initial pilot implementation with IPA partnership" }, { "id": "pilot-deped", "name": "Pilot - DepEd Arm", "color": "#49ac57", "description": "Initial pilot implementation with DepEd partnership" }, { "id": "scaleup1", "name": "Scale-Up Round 1", "color": "#f26529", "description": "First scale-up implementation phase" }, { "id": "scaleup2", "name": "Scale-Up Round 2", "color": "#2b4085", "description": "Second scale-up implementation phase" }, { "id": "scaleup3", "name": "Scale-Up Round 3", "color": "#8e44ad", "description": "Third scale-up implementation phase" } ] }
Field Explanations:
  • id: Unique identifier (lowercase, no spaces)
  • name: Display name that appears in filters and stats
  • color: Hex color code for markers and UI elements
  • description: Optional description for documentation

Step 2: Update Your Excel File

Add a new column to your Excel file with the exact name from the config:

School Name SDO Coordinates Pilot - IPA Arm Pilot - DepEd Arm Scale-Up Round 1 Scale-Up Round 2 Scale-Up Round 3
School A SDO1 14.65,120.96 Yes No No Yes Yes
School B SDO2 14.66,120.97 No Yes Yes No No

Step 3: Convert to schools.json

  1. Go to the Map File Converter page
  2. Upload your updated Excel file
  3. The converter will automatically read the new configuration
  4. Download the generated schools.json file
  5. Replace the old schools.json file

Step 4: Refresh the Implementation Map

  1. Open the Implementation Map page
  2. The new round will automatically appear in:
    • Filter checkboxes with the correct color
    • Statistics panel
    • Map markers and clusters
    • School popup information

Troubleshooting

Round Not Appearing?

  1. Check that the name in rounds-config.json exactly matches the Excel column header
  2. Ensure the JSON syntax is valid (use a JSON validator)
  3. Check browser console for error messages

Colors Not Showing?

  1. Verify color codes are valid hex format (#xxxxxx)
  2. Clear browser cache and refresh

Schools Not Updating?

  1. Make sure you replaced the old schools.json file
  2. Check that the Excel file has "Yes"/"No" values in the new column
  3. Verify the Map File Converter processed the file without errors
Important: Always backup your existing files before making changes, and test with a small sample first!
